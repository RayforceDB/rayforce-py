# :material-card-text: String

The `String` type represents a [:material-vector-line: Vector](./vector.md) of [:material-format-letter-case: Char](./char.md).

## Usage

```python
>>> from rayforce import String

>>> name = String("Hello, World!")
>>> name
String(Hello, World!)  # type: rayforce.String

>>> name.value
'Hello, World!'  # type: str
```
